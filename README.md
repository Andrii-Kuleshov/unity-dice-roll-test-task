Тестове завдання Unity.
Тестове завдання для Otaman Games.

Опис:
Реалізовано просту механіку кидка кубика у стилі DnD.

Логіка роботи:
Генерується випадкове число від 1 до 20;
Отриманий результат порівнюється зі складністю перевірки;
Якщо результат більший за складність — успіх;
Якщо результат менший або рівний складності — провал.

Приклад:
Кидок кубика: 18
Модифікатор: 3
Загальний результат: 21
Результат: Успіх

Версія Unity:
Unity 6 (6000.3.10f1).

Розташування скрипта:
Assets/Scripts/DiceRoll.cs
